Download a portable version from the Releases over there-->

# What does it do?
It allows you to create notes and add drawings to them or image attachments.
It auto focuses on the active note to help you concentrate maaaaan.

# What doesn't it do?
Unlike like most note taking apps it doesn't handle storing your notes on the "cloud" or "syncing" notes.
You just point it at a local folder and that is what it uses to store/display your notes.
* if you want to sync your notes across multiple computers or store notes on the "cloud" just use a notes folder thats in Dropbox/Google Drive/OneDrive/Syncthing/Network etc
  * when syncing multiple computers that have Noat Boat actively running use the Refresh notes button to sync notes from the other active computer
